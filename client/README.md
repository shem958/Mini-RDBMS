# Mini RDBMS Project

## Backend
1. Install dependencies: `npm install`
2. Start REPL: `npm run dev`
3. Start server: `npm run server`

## Frontend
1. `cd client && npm install`
2. `npm start` to run React app

## Features
- Persistent JSON-based mini-RDBMS
- SQL-like parser (CREATE, INSERT, SELECT, DELETE)
- Primary & Unique keys
- In-memory indexes
- Interactive REPL
- React frontend demo
